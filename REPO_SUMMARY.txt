🚀 QUICK START
═══════════════════════════════════════════════════════════════════════════

1. Initialize Git Repository
   $ cd fleetos
   $ git init
   $ git add .
   $ git commit -m "Initial commit: FleetOS"

2. Create GitHub Repository
   - Go to https://github.com/new
   - Name: "fleetos"
   - Push to GitHub

3. Setup Local Development
   $ python -m venv venv
   $ source venv/bin/activate
   $ pip install -r requirements.txt
   $ cp .env.example .env

4. Start Services
   $ docker-compose up -d

5. Test CLI
   $ python -m fleetos.cli --help
   $ python -m fleetos.cli run "My test command"
